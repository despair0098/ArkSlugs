# ArkSlugs

Arkslugs is a simulator game that uses Random Number Generator, TFT LCD, and
Servo Motor to play the game. The aim of this game is to raise your slugs to rank S and
it can change the appearance. It also has battle moments to raise your slug to higher
rank. You use the joystick to do actions which includes training, feeding and playing.
You can all of these actions to raise your slugs. The LCD screen gives 4 different actions for you to take.
